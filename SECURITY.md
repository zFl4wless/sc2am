# Security Policy

## Supported Versions

We only provide security fixes for the latest stable release line.

| Version | Supported          |
|---------|--------------------|
| 1.x     | :white_check_mark: |
| < 1.x   | :x:                |

## Reporting a Vulnerability

If you believe you have found a security vulnerability in `sc2am`, please do not open a public issue.

Please report it privately through GitHub Security Advisories:
- Go to the repository on GitHub
- Open the **Security** tab
- Click **Report a vulnerability**

If GitHub Security Advisories are not available for this repository, please contact the maintainers through a private channel instead.

Please include:
- A short description of the issue
- The affected version(s)
- Steps to reproduce, if possible
- Any relevant logs, screenshots, or proof of concept details

We will review reports as quickly as possible and aim to acknowledge receipt within a few business days. If the report is confirmed, we will work on a fix and coordinate disclosure before publishing it.
